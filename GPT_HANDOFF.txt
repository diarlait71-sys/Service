New optional columns recently added:
- Профиль весов
- Профиль логики

Meaning of new profile columns:
- If the same profile value is used in multiple rows, one common weight/logic profile is applied to all those dealer centers.
- If a profile value is unique on one row, it works as a dealer-center-specific setup.
- If the profile column is empty, the app falls back to the older auto behavior.

10. WEIGHT FILE RESOLUTION LOGIC
Implemented in real_bonus_calculator.py.

Weights are resolved in this order:
1. Use profile from column Профиль весов, if present.
2. Try direct file: {Профиль весов}.xlsx
3. Try prefixed file: Доля показателей для {Профиль весов}.xlsx
4. Try match by position name.
5. Fallback to Доля показателей для Руководителя отдела сервиса.xlsx or first matching file.

Recommended naming for grouped profiles:
- Доля показателей для ROS_GROUP_1.xlsx
- Доля показателей для ALMATY_SPECIAL.xlsx

11. LOGIC FILE RESOLUTION LOGIC
Implemented in real_bonus_calculator.py.

Logic is resolved in this order:
1. Use profile from column Профиль логики, if present.
2. Try direct file: {Профиль логики}.xlsx
3. Try Формула расчета - {Профиль логики}.xlsx
4. Try Формула расчета {Профиль логики}.xlsx
5. Fallback to Формула расчета.xlsx

Recommended naming:
- Формула расчета - ROS_GROUP_1.xlsx
- Формула расчета - ALMATY_SPECIAL.xlsx

14. CORE BONUS CALCULATION RULES CURRENTLY USED
Base salary:
- reper * weight of Оклад (База)

Services bonus:
- reper * weight * (fact / plan)

Spare parts bonus:
- reper * weight * (fact / plan)

Marginality bonus:
- full metric weight is paid only if actual marginality >= planned marginality

Negliquidity:
- negative motivator; applied when fact > plan

Other KPIs:
- evaluated in _calculate_other_metric_bonus()
- currently supports text comparison, yes/no logic, and number inequality logic

15. SOURCE OF PLAN / FACT DATA CURRENTLY USED
Services:
- plan and fact are summed for selected dealer center from Факт Услуги.xlsx

Spare parts:
- plan and fact are summed for selected dealer center from Факт Запасные части.xlsx

Marginality:
- per dealer center from Факт маржанальность.xlsx
- formula: total margin / total revenue

Other KPIs:
- plan from план по остальным показателям.xlsx
- fact from dealer center column in Факт по остальным показателям.xlsx when available
